# **App Name**: Modulares GM

## Core Features:

- Dynamic Content Display: Showcase company information, services, statistics, and brand partnerships on the public-facing website, ensuring all data is easily manageable by administrators.
- User Contact & Lead Submission: Provide a contact form for potential customers to submit inquiries, which are then saved for lead management in the admin dashboard.
- Product E-commerce Catalog: Enable users to browse and view detailed information, including images, descriptions, and pricing, for all available furniture products in a dedicated online store section.
- Shopping Cart & Secure Checkout: Allow customers to add products to a shopping cart, adjust quantities, and complete purchases through a secure multi-step checkout process with various payment options. Order data is persisted in a database.
- Comprehensive Admin Dashboard (CMS): A secure and intuitive interface for administrators to manage all website content, including hero section, services, product listings, brands, contact details, and SEO metadata.
- AI-Powered Sales & Content Tools: Integrate generative AI tools within the admin panel to assist with lead analysis (intent extraction), formal proposal generation, SEO meta tag optimization, product description drafting, and personalized order update emails. These tools function as smart assistants to enhance operational efficiency.
- Interactive AI Chatbot: A public-facing chatbot tool powered by generative AI that can answer user queries regarding Modulares GM's services, products, and policies, providing immediate assistance and engagement.

## Style Guidelines:

- Primary brand color: A vibrant teal (#0A858C) used for key branding elements, main call-to-action buttons, and interactive highlights.
- Secondary brand color: A warm, earthy gold (#C5985F) used for subtle accents, specific tags, and status indicators.
- General palette (Dark Mode): Utilizes a deep charcoal blue-gray background (#0D1318), contrasting with white text for high readability.
- General palette (Light Mode): Features a soft, off-white background (#F8FAFC), paired with dark blue-gray text (#19242D) for a clean, professional look.
- Headline font: 'Playfair Display' (serif), used for prominent titles and decorative elements, offering an elegant and high-end feel.
- Body and functional text font: 'Inter' (sans-serif), ensuring readability and a modern, objective feel across all informational and interactive elements.
- Utilize a consistent set of clean, modern line-art icons from the Lucide-React library to maintain a contemporary and clear visual language throughout the application.
- Implement a modular and responsive grid-based layout for optimal viewing across various devices, focusing on generous spacing, clear content hierarchy, and a clean card-based design for content presentation.
- Incorporate subtle, tasteful animations for page transitions, interactive elements (buttons, hover states), and data loading indicators, enhancing user experience without distracting from content.
